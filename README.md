# Sentrix

Cyber insurance marketing site with a live quote wizard and plan comparison.

**Tagline:** Cyber Insurance for Modern Businesses

## Stack

- React 19 + TypeScript
- Vite 8
- Tailwind CSS v4
- shadcn/ui (Radix primitives)
- Framer Motion (`motion/react`)
- Zustand (quote state)
- TanStack React Query
- React Router + react-helmet-async (SEO)

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Routes

| Path | Page |
|------|------|
| `/` | Landing — hero, stats, coverage, FAQ |
| `/quote` | 27-step cyber health check wizard |
| `/plans` | Plan comparison with live premium slider |

## Scripts

- `npm run dev` — development server
- `npm run build` — production build
- `npm run preview` — preview production build

## Design

Apple-inspired SaaS aesthetic: generous whitespace, subtle motion, frosted navigation, depth via layered shadows, alternating section backgrounds, and Inter typography with clear hierarchy. Brand navy `#061A40` and electric blue accents are preserved throughout.
